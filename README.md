# Quotero Core

Quotero Core is a SaaS platform for building and managing AI chatbots. It enables clients to upload documents, create chatbots with custom configurations, and track API usage through a dashboard interface.


## Tech Stack

- **Framework**: Next.js 16
- **Language**: TypeScript
- **Database**: PostgreSQL with Prisma ORM
- **Authentication**: Clerk
- **Styling**: Tailwind CSS
- **Monorepo**: Turborepo with pnpm workspaces
- **UI Components**: Custom component library built on top of Shadcn
